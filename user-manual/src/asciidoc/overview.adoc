== Overview

=== Accessing Swagger UI

Most services expose Swagger UI for API exploration.

Run locally (check the application config when in doubt):

- **Card Terminal Client**: http://localhost:80/swagger-ui/index.html
- **VSDM Client**: http://localhost:80/swagger-ui/index.html

As part of docker compose:

- **ZeTA PDP PoPP**: http://localhost:9112/swagger-ui/index.html
- **PoPP Server**: http://localhost:9210/swagger-ui/index.html
- **VSDM Server**: http://localhost:9130/swagger-ui/index.html
